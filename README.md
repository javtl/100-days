# C++ Learning Journey: 2024 Foundations

A chronological record of my progress mastering **C++** fundamentals. This repository serves as a technical log of my transition from basic logic to advanced procedural programming and system configuration.

## 🚀 Overview
This project documents my daily practice during early 2024, focusing on memory management, algorithm efficiency, and clean code practices in C++. All exercises were developed using **Visual Studio Code** and compiled with **GCC/G++**.

## 🛠️ Tech Stack
* **Language:** C++ (ISO C++17/20)
* **Compiler:** G++
* **IDE:** Visual Studio Code
* **Debugger:** GDB

## 📈 Learning Timeline (January 2024)

| Date | Focus Area | Key Concepts Covered |
| :--- | :--- | :--- |
| **Jan 01-04** | **Fundamentals** | Conditionals, Loops (`for`, `do-while`), and Flow Control. |
| **Jan 05** | **Functions** | Pass-by-value vs. Pass-by-reference (`&`), Prototypes. |
| **Jan 06** | **Intro to OOP** | Classes, Access Specifiers (`public`/`private`), Constructors. |
| **Jan 07-09** | **Applied Logic** | `switch` statements, `string` manipulation, `bool` functions. |
| **Jan 10-11** | **Modern Tooling** | STL Libraries (`cmath`, `iomanip`), VS Code environment setup. |

| **Jan 17-20** | **Advanced Basics** | `while` loops, `typedef` for abstraction, Matrix manipulation. |

## 💡 Key Highlights
* **Algorithmic Thinking:** Implementation of sorting and searching algorithms from scratch.
* **Memory Awareness:** Practical use of pointers and references to optimize performance.
* **Clean Code:** Focus on modularity by breaking down complex problems into reusable functions.
